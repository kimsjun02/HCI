## 제품 개요

**스마트 일정 관리 시스템(Smart Task Calendar)**은 메일 기반의 자동 할일 분석 기능을 탑재한 웹 어플리케이션입니다.
사용자가 받은 이메일에서 자동으로 할일을 추출하여 캘린더에 등록하고, 이를 통해 체계적인 일정 관리를 돕는 AI 기반의 생산성 도구입니다.

### 주요 특징

- **이메일 자동 분석**: 받은 메일에서 할일 자동 추출
- **캘린더 통합**: 추출된 할일을 캘린더에 자동 등록
- **사용자 확인 알림**: 자동 등록된 할일의 정확성 검증
- **직관적 일정 관리**: 할일의 추가, 수정, 삭제 기능

---

## 제품 비전

### 목적

대학원생과 같은 바쁜 직업인들이 **이메일에서 놓칠 수 있는 중요한 할일을 자동으로 포착**하여,
체계적인 캘린더 기반 일정 관리가 가능하도록 하는 것입니다.

### 해결하려는 문제

| 문제                                                      | 영향                                 | 해결 방안                                   |
| --------------------------------------------------------- | ------------------------------------ | ------------------------------------------- |
| 매일 많은 이메일을 받지만 모든 메일을 즉시 확인할 수 없음 | 중요한 할일 누락으로 인한 업무 지연  | AI 기반 이메일 분석으로 할일 자동 추출      |
| 메일과 캘린더가 분리되어 있어 일정 관리가 산발적          | 통합된 일정 관리의 어려움            | 메일에서 추출한 할일을 캘린더에 자동 동기화 |
| 자동 등록된 할일의 정확도에 대한 불신                     | 사용자가 자동 기능을 신뢰하기 어려움 | 등록 전 사용자 확인 및 승인 프로세스        |

- **차별점**: 메일 기반 자동 할일 분석 기능 추가

- **참고 점**: 할일의 우선순위 및 카테고리 관리, 반복 일정 처리
- **이유**: 체계적인 할일 관리 방식을 학습하여, 사용자가 복잡한 일정을 효과적으로 분류할 수 있도록 지원
- **차별점**: 이메일 기반의 자동 할일 생성으로 수동 입력 과정 제거

- **참고 점**: 이메일과 캘린더의 원활한 통합, 임박한 일정 알림 시스템
- **이유**: 업무용 이메일과 캘린더의 자연스러운 연결 방식을 참고하여, 메일-일정 간 통합도 향상
- **차별점**: 더 높은 수준의 자동화 (AI 기반 할일 분석)

## 핵심 기능 및 상세 사양

### 1. 할일 관리 시스템

#### 1.1 할일 추가 (Create)

- **사용자 가치**: 사용자가 직접 할일을 추가하거나 이메일에서 자동으로 추출된 할일을 캘린더에 등록할 수 있습니다.
- **상세 사양**:
  - 제목, 설명, 마감 날짜/시간, 우선순위 입력 가능
  - 할일을 이메일과 연결하여 컨텍스트 제공
  - 반복 일정 설정 지원 (매일, 매주, 매월)

#### 1.2 할일 수정 (Update)

- **사용자 가치**: 등록된 할일을 언제든 수정할 수 있어, 일정 변경에 유연하게 대응할 수 있습니다.
- **상세 사양**:
  - 마감 날짜, 우선순위, 설명 수정 가능
  - 버전 관리로 이전 변경 이력 조회 가능

#### 1.3 할일 삭제 (Delete)

- **사용자 가치**: 완료되었거나 더 이상 필요 없는 할일을 제거하여 캘린더를 정리된 상태로 유지합니다.
- **상세 사양**:
  - 개별 할일 또는 반복 일정 전체/일부 삭제 가능
  - 삭제 전 확인 프로세스 제공

#### 1.4 할일 상태 관리

- **사용자 가치**: 할일의 진행 상황을 추적하여 우선순위 있는 일정 관리가 가능합니다.
- **상세 사양**:
  - 상태: 미시작 → 진행 중 → 완료
  - 각 상태별 필터링 및 검색 지원

---

### 2. 캘린더 보기

#### 2.1 다양한 캘린더 뷰

- **사용자 가치**: 월간, 주간, 일간 뷰를 제공하여 사용자가 원하는 시간 단위로 일정을 확인할 수 있습니다.
- **상세 사양**:
  - **월간 뷰**: 전체 월별 일정 한눈에 파악
  - **주간 뷰**: 시간대별 일정 및 할일 세부 사항 확인
  - **일간 뷰**: 오늘의 할일에 집중할 수 있는 상세 보기
  - 뷰 전환 시 자동으로 필터링된 일정 표시

#### 2.2 일정 시각화

- **사용자 가치**: 색상 코딩과 아이콘을 통해 할일의 중요도와 상태를 한눈에 파악할 수 있습니다.
- **상세 사양**:
  - 우선순위별 색상 구분 (긴급: 빨강, 중요: 주황, 일반: 파랑)
  - 마감 기한이 임박한 할일 강조 표시
  - 완료된 할일 투명도 처리로 시각적 구분

---

### 3. 이메일 기반 자동 할일 분석

#### 3.1 이메일 연결 및 분석

- **사용자 가치**: 이메일을 읽을 필요 없이 시스템이 자동으로 할일을 찾아내므로, 사용자는 집중력을 잃지 않고 중요한 일에만 집중할 수 있습니다.
- **상세 사양**:
  - Gmail, Outlook 등 주요 메일 서비스 연동
  - 자연어 처리(NLP)를 통한 할일 자동 추출
  - 메일 발신자, 제목, 본문에서 마감 기한 자동 추출
  - 메일의 긴급도를 분석하여 우선순위 자동 설정

#### 3.2 AI 기반 할일 추출 로직

- **사용자 가치**: 메일의 맥락을 이해하여 정확한 할일을 추출하므로, 사용자의 수동 작업을 최소화합니다.
- **상세 사양**:
  - 동작 동사 감지 (e.g., "~해주세요", "~해야 합니다", "~확인 부탁")
  - 시간 표현 추출 (e.g., "내일까지", "이번 주말", "5월 30일")
  - 다중 할일 인식 (한 메일에 여러 할일이 포함된 경우)

---

### 4. 사용자 확인 알림 시스템

#### 4.1 할일 추가 시 알림

- **사용자 가치**: 자동으로 추가된 할일을 검토할 기회를 제공하여, 사용자는 시스템을 신뢰하고 사용할 수 있습니다.
- **상세 사양**:
  - 새로운 할일이 캘린더에 추가될 때 브라우저 알림 및 인앱 알림 제공
  - 알림에서 할일 미리보기 제공
  - "수락", "거부", "수정 후 수락" 옵션 제공

#### 4.2 마감 기한 임박 알림

- **사용자 가치**: 마감 기한을 놓치지 않도록 사전 알려주므로, 중요한 업무를 시간 내에 완료할 수 있습니다.
- **상세 사양**:
  - 마감 1주일 전: 일일 요약 알림
  - 마감 24시간 전: 강조된 알림
  - 마감 시간 초과: 빨강 표시 및 재알림

## 제안하는 추가 기능

### 우선순위 제안 기능

- **설명**: AI가 메일의 내용과 발신자 정보를 바탕으로 할일의 우선순위를 제안
- **사용자 가치**: 중요한 일을 자동으로 구분하여 사용자의 의사결정 시간을 단축

### 반복 일정 자동 생성

- **설명**: "매주 월요일 회의", "매월 보고서 제출" 같은 반복 패턴을 인식하여 자동 등록
- **사용자 가치**: 반복되는 일정을 직접 설정하지 않아도 자동으로 관리 가능

### 협업 기능

- **설명**: 팀원들과 캘린더와 할일을 공유하고, 코멘트 및 진행 상황 공유
- **사용자 가치**: 그룹 프로젝트나 팀 협업 시 모두가 같은 정보를 기반으로 작업

### 분석 대시보드

- **설명**: 주간/월간 생산성 통계 제공 (완료율, 마감 초과율, 카테고리별 분포)
- **사용자 가치**: 자신의 일정 관리 패턴을 파악하고 개선 방향 도출 가능

### 다크모드

- **설명**: 밤시간 사용 시 눈 피로도 감소
- **사용자 가치**: 늦은 밤까지 일하는 대학원생의 사용성 향상

## 사용자 시나리오

### 시나리오 1: 메일 기반 자동 할일 추출

**"태형의 일상 속 자동 할일 관리"**

태형은 아침 9시에 랩실에 도착합니다. 연구실 컴퓨터를 켜자마자 스마트 일정 관리 시스템에서 새로운 알림이 뜹니다.
지도교수로부터 받은 "재료 특성 분석 결과를 이번 주 금요일까지 제출해주세요"라는 메일이 자동으로 할일로 변환되어 캘린더에 추가된 것입니다.
태형은 인앱 알림에서 해당 할일을 빠르게 검토하고 "수락"을 누릅니다.
이제 그 할일은 태형의 캘린더 금요일에 자동으로 고정되어, 다른 일정과 충돌하지 않는지 확인할 수 있습니다.

### 시나리오 2: 마감 기한 관리

**"임박한 마감 기한 알림으로 업무 빠뜨리지 않기"**

수요일 오후, 태형이 연구에 집중하고 있을 때 시스템에서 알림이 옵니다.
"재료 특성 분석 결과 제출까지 1일 남음"이라는 메시지와 함께 할일이 강조 표시됩니다.
태형은 마감을 놓치지 않고, 목요일 저녁까지 결과를 정리할 시간을 확보할 수 있습니다.

## 결론

**스마트 일정 관리 시스템**은 바쁜 대학원생과 직업인들이 메일에서 놓칠 수 있는 중요한 할일을 자동으로 포착하여,
